# Ask
 A basic, no frills FAQ built with jquery.
 
 ## Browser Compatibility
 Ask has been tested, and it works on Opera 74.0.3911.203, Chrome 88.0.4324.192, Safari 13.1.3, Internet Edge 88.0.705.81, and Firefox 86.0.
